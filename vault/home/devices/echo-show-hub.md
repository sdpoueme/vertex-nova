# Echo Show — Primary Hub

Main smart home hub. All Alexa-compatible devices connect through this.

## Connected Systems

- [[echo-devices]] — All Echo speakers throughout the house
- [[sonos-system]] — Sonos speaker system
- [[fire-tv-devices]] — Fire TV sticks/devices
- Smart home devices (lights, locks, thermostats, cameras, etc.)

## Capabilities

- Voice control for all connected devices
- Visual display for cameras, recipes, video calls
- Smart home dashboard
- Intercom / Drop In between Echo devices
- Home assistant channel (via Alexa custom skill)
