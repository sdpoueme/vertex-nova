# Home Status

Give the user a quick overview of their home:

1. Check today's daily note for pending tasks (`- [ ]`)
2. Search for home events from the past 7 days (tag: type/home-event)
3. Search for tasks with upcoming `next_due` dates (within next 14 days)
4. Search for any open issues (events without resolution)
5. Check for devices with warranties expiring soon

Present a concise summary:
- 🏠 **Home Status** — date
- **Pending tasks** — count and brief list
- **Recent events** — last 3-5 events
- **Upcoming maintenance** — items due in next 2 weeks
- **Open issues** — unresolved problems

Keep it brief and actionable. Highlight anything urgent.
